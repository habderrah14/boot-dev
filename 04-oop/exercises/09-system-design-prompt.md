# Module 04 · Extended Assessment — System Design Prompt

## 1) Warm-up

Define boundaries, interfaces, extension points, and substitution safety.

## 2) Standard

Design an object model for a small backend domain with clear responsibilities.

## 3) Bug hunt

Identify three design smells and mitigations.

## 4) Stretch

Write a 250-word memo on composition vs inheritance tradeoffs.

## 5) Stretch++

Peer critique and require one architecture improvement.

**Takeaway:** Good object models reduce future change cost.
