# Module 02 · Extended Assessment — Code Review Task

## 1) Warm-up

Create a 6-point review rubric for shell scripts and command safety.

## 2) Standard

Review one script and leave 5 actionable comments: issue → impact → fix.

## 3) Bug hunt

Find three risks: unsafe globbing, missing quoting, destructive command without guard.

## 4) Stretch

Write approval memo (blockers vs non-blockers).

## 5) Stretch++

Do reciprocal review with a peer and resolve one disagreement with a test.

**Takeaway:** Script reviews prevent operational outages.
