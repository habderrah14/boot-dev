# Module 03 · Extended Assessment — Code Review Task

## 1) Warm-up

Build a 6-point rubric for reviewing commit hygiene and PR history quality.

## 2) Standard

Review one PR history and leave 5 actionable comments.

## 3) Bug hunt

Find three risks: oversized commit, unclear message, dangerous force push workflow.

## 4) Stretch

Write go/no-go memo with required branch policy improvements.

## 5) Stretch++

Pair-review and resolve one disagreement with a reproducible workflow test.

**Takeaway:** Good Git review lowers integration risk.
