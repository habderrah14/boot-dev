# Module 01 — Learn to Code in Python

> Your beginner runway. This module is now structured to help you move one Boot.dev lesson at a time without being pulled into future concepts too early.

## Map to Boot.dev

This module parallels the Boot.dev course **"Learn to Code in Python"**.

## Learning outcomes

By the end of this module you will be able to:

- Read and write Python fundamentals with confidence.
- Explain variables, functions, scope, loops, and core collections in plain language.
- Debug beginner-level bugs and verify fixes.
- Build one cumulative CLI project through gradual checkpoints.

## Study path (default flow)

For each chapter:

1. **Read now:** chapter file (`NN-topic.md`).
2. **Practice now:** matching exercise file (`exercises/NN-topic-exercises.md`) in L1 → L4 order.
3. **Build now:** update one checkpoint in the progressive mini-project track.
4. **Optional later:** deeper sections and cross-module links.

## Progressive mini-project track

Use a single cumulative project for this module:

- [Python Fundamentals Companion Track](mini-projects/00-python-fundamentals-companion-track.md)

Checkpoint mapping for pilot lessons:

- After Chapter 02 (Variables) → Checkpoint A
- After Chapter 03 (Functions) → Checkpoint B
- After Chapter 04 (Scope) → Checkpoint C
- After Chapter 08 (Loops) → Checkpoint D

## Prerequisite matrix (pilot)

- **Variables** requires: Introduction only.
- **Functions** requires: Variables.
- **Scope** requires: Functions.
- **Loops** requires: Functions (which already depends on Variables) + Comparisons.

Exercises in these chapters should not require future lessons.
