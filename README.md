# The Backend Developer's Companion

A guided companion to the [Boot.dev Backend Developer Path (Python / TypeScript)](https://www.boot.dev/paths/backend?tech=python-typescript), optimized for **beginners learning lesson-by-lesson**.

## Target mode

Use this repo when your current need is:

- “I am on this Boot.dev lesson right now.”
- “I need a simple explanation, a visual, and practice that matches my current level.”
- “I want one project track that grows with me, not many disconnected projects.”

## Companion goals

This repo now prioritizes:

- **Beginner-first language.** Start simple, add depth later.
- **Visual-first explanations.** Key ideas should have traceable diagrams.
- **Gradual exercises.** Every exercise set should move from basic to stretch.
- **One mini-project track.** Module 01 uses one cumulative track with checkpoints.
- **No placeholder noise.** Learner-irrelevant or duplicate artifacts are retired.

## Not now (optional detours)

If you are early in the path, you can skip these until needed:

- Deep optimization and advanced trade-off analysis.
- Appendix system design drills.
- Cross-module integration projects.

## How to use this companion

1. Do the Boot.dev lesson.
2. Read the matching chapter here.
3. Complete the matching exercise levels (L1 → L4).
4. Advance one checkpoint in the module's mini-project track.
5. Take the quiz.

Start with [Module 01 study path](01-python/README.md#study-path-default-flow).
