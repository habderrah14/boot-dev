# Module 04 · Extended Assessment — Code Review Task

## 1) Warm-up

Create a rubric for class design, cohesion, coupling, and API clarity.

## 2) Standard

Review one solution and leave 5 actionable comments.

## 3) Bug hunt

Find three risks: god class, inheritance misuse, weak encapsulation.

## 4) Stretch

Write approval memo with blockers/non-blockers.

## 5) Stretch++

Reciprocal review and resolve one disagreement with tests.

**Takeaway:** OOP review should optimize maintainability and correctness.
