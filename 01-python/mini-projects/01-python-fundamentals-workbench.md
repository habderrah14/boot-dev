# Mini-project — Python Fundamentals Workbench

## Goal

Build a single CLI script that demonstrates variables, functions, loops, lists, dicts, sets, and structured error handling.

## Acceptance criteria

- Includes at least 5 commands or modes.
- Uses one function per command path.
- Stores data using at least 3 core structures (list/dict/set).
- Includes input validation with clear error messages.
- Has a minimal `unittest` suite covering happy and error paths.

## Hints

- Start with a command dispatcher dict.
- Keep business logic pure where possible.
- Add one failing test before each bug fix.

## Stretch

- Add persistence to JSON file.
- Add argparse-based CLI flags.
- Add a simple benchmark for one operation.
