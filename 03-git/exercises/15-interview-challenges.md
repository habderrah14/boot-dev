# Module 03 · Extended Assessment — Interview Challenges

## 1) Warm-up

Answer five Git diagnosis prompts (merge, rebase, reset, reflog, bisect).

## 2) Standard

Solve two timed Git scenario challenges and explain tradeoffs.

## 3) Bug hunt

Reproduce and fix one broken-history scenario safely.

## 4) Stretch

Explain chosen recovery strategy for a non-expert teammate.

## 5) Stretch++

Run a mock and log one improvement for next attempt.

**Takeaway:** Interview Git is about safe recovery logic.
