# Module 11 · Extended Assessment — Code Review Task

## 1) Warm-up — SQL review rubric

Define an 8-point rubric: schema correctness, constraints, transaction safety, index strategy, query clarity, security, migration safety, observability.

## 2) Standard — review one solution slice

Review one file from `11-sql/solutions/` and leave 6 actionable comments in `practice/13-code-review-task/review-notes.md`.

## 3) Bug hunt — risk-oriented findings

Find and patch three risks:

- missing constraint,
- unsafe migration step,
- query plan blind spot.

## 4) Stretch — approval memo (≤250 words)

Write production approval decision: blockers vs non-blockers.

## 5) Stretch++ — reciprocal review

Exchange SQL snippets with a peer and log one resolved disagreement.

**Takeaway:** SQL review quality is measured by incident prevention, not formatting.
