# Module 16 — Learn How to Find a Programming Job

> The last module is not about code. It's about packaging a year of work into something a hiring manager will read in 30 seconds — and into a 45-minute conversation that convinces them to hire you.

## Map to Boot.dev

Parallels Boot.dev's **"Learn How to Find a Programming Job"** course, chapter-for-chapter.

## Learning outcomes

By the end of this module you will be able to:

- Build a job-hunt strategy proportional to the market you're targeting.
- Choose portfolio projects that speak to the roles you want.
- Craft a GitHub profile, resume, and LinkedIn that collectively pass the recruiter screen.
- Apply at volume without burning out.
- Network in a way that does not feel transactional.
- Prepare for and perform in technical interviews (coding, systems, behavioral).
- Decide intelligently about relocation and remote-first roles.

## Prerequisites

- All earlier modules — you're now the candidate the earlier modules described.
