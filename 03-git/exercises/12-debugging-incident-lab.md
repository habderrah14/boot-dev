# Module 03 · Extended Assessment — Debugging Incident Lab

## 1) Warm-up

Classify three Git failure symptoms and choose first diagnostic command.

## 2) Standard

Write a postmortem for one branch/merge/rebase incident.

## 3) Bug hunt

Recover from detached HEAD, bad rebase, and accidental reset.

## 4) Stretch

Draft a runbook for "history rewrite on shared branch".

## 5) Stretch++

Simulate incident response with a peer and improve runbook.

**Takeaway:** Version-control incidents are recoverable with calm, traceable steps.
