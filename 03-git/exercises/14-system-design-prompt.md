# Module 03 · Extended Assessment — System Design Prompt

## 1) Warm-up

Define trunk-based flow, release branches, CI gates, and rollback strategy.

## 2) Standard

Design a team Git workflow for 20 contributors and daily deploys.

## 3) Bug hunt

Identify three bottlenecks and propose process/tooling fixes.

## 4) Stretch

Write a short scaling memo: from 5 to 50 engineers.

## 5) Stretch++

Peer-critique workflow and add one resilience control.

**Takeaway:** Repo workflow design is systems design for teams.
